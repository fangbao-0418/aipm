# 二、顶部工具栏（TopBar）

## UI结构

```text
[Logo] 项目名称

[+ 新建] [AI生成] [保存] [撤销] [重做]

| 工具区 |
[选择] [手型] [文本] [矩形] [组件]

| 视图区 |
[缩放 100%] [适配屏幕]

| 操作区 |
[预览] [导出代码] [分享]
```

***

## 交互说明（关键）

| 操作    | 行为       |
| ----- | -------- |
| AI生成  | 打开AI输入弹窗 |
| 撤销/重做 | 操作历史     |
| 预览    | 全屏原型     |
| 导出    | 打开导出弹窗   |

# 三、左侧面板（Sidebar）

## 三个 Tab（必须有）

```text
[图层] [组件] [资源]
```

***

## 3.1 图层（Layers）

```text
页面1
 ├─ Header
 ├─ Table
 │   ├─ Row1
 │   ├─ Row2
 ├─ Button
```

👉 支持：

* 拖拽排序

* 隐藏/显示 👁

* 锁定 🔒

* 重命名

***

## 3.2 组件库（Components）

```text
基础组件
  - Text
  - Image
  - Button
  - Input

业务组件
  - 商品卡片
  - 用户卡片
  - 数据卡片

Antd组件
  - Table
  - Form
  - Modal
```

👉 交互：

```text
拖拽 → 画布
点击 → 插入默认位置
```

***

## 3.3 资源（Assets）

```text
素材组件
模板
图标
图片
```

👉 来源：

* 用户保存的组件

* 系统模板

* AI生成组件


