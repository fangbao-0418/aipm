# 图片后处理（Color Adjustments）

# 一、Sketch 颜色调整完整对应关系

你截图 UI：

```txt
色相 13%
饱和度 0%
亮度 8%
对比度 0%
```

对应 JSON：

```json
"colorControls": {
  "isEnabled": true,
  "brightness": 0.0799999982118607,
  "contrast": 1,
  "hue": 0.41642466370159,
  "saturation": 1
}
```

---

# 二、字段真实含义（非常关键）

## 1. brightness

```json
"brightness": 0.08
```

对应：

```txt
亮度 +8%
```

会让图片：

* 发白
* 更亮
* 灰度提升

你图里明显：

# 图片整体泛白

就是它。

---

## 2. hue

```json
"hue": 0.41642466370159
```

这个：

# 不是 13%

Sketch 内部：

```txt
0 ~ 1
```

对应：

```txt
0° ~ 360°
```

所以：

```txt
0.4164 × 360 ≈ 150°
```

但是 Sketch UI 会显示成：

```txt
13%
```

因为：

# Sketch UI 和底层不是线性映射

---

它会：

# 改变整体色相

所以：

你原本图片可能不是紫蓝色。

但经过 hue rotate：

变成了：

```txt
蓝紫色调
```

---

## 3. saturation

```json
"saturation": 1
```

注意：

Sketch：

```txt
1 = 原始值
```

不是：

```txt
100%
```

---

计算方式：

```txt
0 = 灰度
1 = 原图
2 = 超高饱和
```

所以你这里：

```txt
饱和度没变
```

---

## 4. contrast

```json
"contrast": 1
```

也是：

```txt
1 = 原始对比度
```

不是：

```txt
100%
```

---

# 三、为什么图片变成了“淡蓝紫”

真正原因是：

# hue + brightness 联合作用

---

## hue

把颜色：

```txt
偏蓝紫
```

---

## brightness

让图片：

```txt
更亮
更白
更虚
```

---

最终：

```txt
梦幻蓝紫风
```

这就是你截图里的效果。

---

# 四、真正影响图片颜色的完整 JSON

这是核心：

```json
{
  "_class": "bitmap",
  "style": {
    "colorControls": {
      "_class": "colorControls",
      "isEnabled": true,

      "brightness": 0.0799999982118607,

      "contrast": 1,

      "hue": 0.41642466370159,

      "saturation": 1
    }
  }
}
```



---

# 五、它在 CSS 里对应什么

本质等于：

```css
filter:
  hue-rotate(150deg)
  brightness(1.08)
  saturate(1)
  contrast(1);
```

# 六、Sketch 颜色调整本质

Sketch：

```txt
colorControls
```

本质就是：

# GPU filter

类似：

```txt
Photoshop 调色
```

不是：

```txt
重新绘制 bitmap
```

---

# 七、你做 Sketch 解析必须支持这个

很多 Sketch 转 HTML：

# 最大问题

就是：

```txt
完全忽略 colorControls
```

结果：

```txt
颜色完全不对
```

尤其：

* 插画
* icon
* AI风格图片
* 模糊渐变图

会：

# 完全失真

---

# 八、你应该这样还原

建议：

```ts
function toCssFilter(colorControls) {
  return `
    hue-rotate(${colorControls.hue * 360}deg)
    brightness(${1 + colorControls.brightness})
    contrast(${colorControls.contrast})
    saturate(${colorControls.saturation})
  `
}
```
